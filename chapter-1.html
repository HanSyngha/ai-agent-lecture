Chapter 01 · 환경 셋업

실습을 돌리려면,
일단 깔자.

파이썬 패키지는 이미 만들어진 기능을 가져오는 일.
오늘 강의 전체에서 쓸 여섯 영역을 한 번에 설치합니다.

·
Why pip install?

남이 만든 코드를,
한 줄로 가져온다.

내 코드는 짧아도 된다. 이미 만들어진 것을 어떻게 엮을지가 전부.
— 이 사고방식이 오늘 강의 전체의 '툴'과 정확히 같다.

패키지 = 외부 함수 차용
·
All-in-one

이 한 줄로,
오늘 쓸 것 전부. 

# Jupyter cell
pip install openai streamlit streamlit_ace openpyxl \
            chromadb pysqlite3-binary python-dotenv \
            ipython ipywidgets --no-cache-dir

--no-cache-dir 는 캐시 폴더 권한이 막힌 환경에서도 안전.
일부가 실패하면 영역별로 쪼개서 재시도하면 됩니다.

·
여섯 영역의 역할

여섯 영역이 모이면
오늘 강의의 골격.

LLM

openai

모든 LLM 호출의 클라이언트. Chapter 3부터 강의 끝까지 매 슬라이드에 등장.

Web UI

streamlit

브라우저에서 챗봇·에이전트를 즉시 띄우기 위한 도구. streamlit_ace 동봉.

Excel

openpyxl

평가 결과를 누적 저장하는 툴이 내부에서 사용. 컬럼은 dict 키 그대로.

RAG

chromadb

벡터 DB로 이력서를 의미 기반 검색. pysqlite3-binary는 호환성용.

Env

python-dotenv

.env 파일 읽기·쓰기. 다음 챕터의 주인공.

Jupyter

ipywidgets

패스워드 입력 위젯 같은 셀 내부 UI. ipython 동봉.

·
Free port 8080

웹앱은
한 포트만
쓸 수 있다.

Streamlit은 8080으로만 뜬다.
이전 셀이 살아 있으면 새 셀은 못 뜬다. 

from utils.helper import terminate_process

terminate_process()
# 8080 점유 프로세스를 종료
terminate_process()는 실습 utils에 미리 포함된 헬퍼. 포트 충돌이 의심되면 가장 먼저 호출.
