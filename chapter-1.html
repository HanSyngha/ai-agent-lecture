← Hub 02
All-in-one install

이 한 줄로
오늘 쓸 거 다 깔립니다.

# Jupyter cell에서 실행 pip install openai streamlit streamlit_ace openpyxl \ chromadb pysqlite3-binary python-dotenv \ ipython ipywidgets --no-cache-dir

--no-cache-dir 캐시 폴더 권한이 막혀 있는 환경에서도 안전. 설치 중 일부 패키지가 실패하면 영역별로 쪼개서 다시 시도하면 됩니다.

← Hub 04
If something fails

영역별 폴백 설치.

LLM 영역
pip install openai \ --force-reinstall --no-cache-dir
웹 UI 영역
pip install streamlit streamlit_ace \ --force-reinstall --no-cache-dir
RAG 영역
pip install chromadb pysqlite3-binary \ --force-reinstall --no-cache-dir
환경변수 영역
pip install python-dotenv \ --force-reinstall --no-cache-dir

--force-reinstall 이미 설치된 버전을 무시하고 재설치. 의존성 충돌이 의심될 때.

← Hub 05
Free port 8080

웹앱은
한 포트만
쓸 수 있다.

실습 환경에서 Streamlit은 8080 포트로만 뜹니다. 이전 셀이 살아 있으면 새 셀은 못 뜹니다.

from utils.helper import terminate_process terminate_process() # 8080 점유 프로세스 종료

terminate_process는 실습 utils에 미리 포함된 헬퍼. 8080 포트를 점유한 프로세스를 찾아서 깔끔하게 종료합니다.
